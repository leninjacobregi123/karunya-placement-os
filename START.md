# KPOS — Placement Preparation OS

## Day 1 in 3 Steps

### Step 1: Clone and set up (one terminal)
```bash
git clone https://git.karunya.edu/ctc/karunya-placement-os.git
cd karunya-placement-os
```

### Step 2: Start studying (open Claude Code in the repo, type `/start`)
```
/start coding
```

Claude Code will: check your Python, create progress tracking, and start Day 1.

### Step 3: Chat with Claude Code
Follow the exercises. Type answers directly in the chat:
- Write Python code → Claude Code runs it
- Get it wrong → Claude Code explains what went wrong (with hints first)
- Get it right → Claude Code advances to the next exercise
- Save progress → Claude Code saves it to `~/.kpos/student.json`, you don't touch it

### Other commands (in Claude Code):
- `/showprogress` — see your scores, weak areas, confidence
- `/hints` — get hints for the current exercise
- `/skip` — skip the current exercise, advance to next
- `/review` — review a previous day's work

---

## What You'll Get in 30 Days

Choose your path when you type `/start`:
- `/start coding` → Coding + DSA (Python, arrays, logic puzzles)
- `/start aptitude` → Aptitude + Reasoning (percentages, ratios, logic)
- `/start full` → Both paths (maximum coverage)
