# Syllogisms — Core Rules

## The Four Statement Types

| Type | Form | Draw |
|---|-|---|
| Universal Affirmative | All A are B | A circle inside B |
| Universal Negative | No A is B | Two separate circles |
| Particular Affirmative | Some A are B | Two overlapping circles |
| Particular Negative | Some A are not B | Part of A outside B |

## The Golden Rule

**"All A are B" does NOT mean "All B are A".** It converts only to "Some B are A."

## Valid Inferences

1. **All + All → All**
   All A are B. All B are C. → All A are C.

2. **All + No → No**
   All A are B. No B are C. → No A are C.

3. **Some + All → Some**
   Some A are B. All B are C. → Some A are C AND Some C are A.

4. **Some + No → Some (reversed)**
   Some A are B. No B are C. → Some A are not C.

5. **Some + Some → NO conclusion** (can't deduce)

6. **No + No → NO conclusion** (can't deduce)

## Conversion Rules

| Original | Valid | Invalid |
|---|-|-|
| All A are B | Some B are A | All B are A |
| No A is B | No B is A | All B are A |
| Some A are B | Some B are A | All B are A |

## Diagram Method

1. Draw each premise on the SAME diagram
2. Test each conclusion against ALL possible valid diagrams
3. If true in all → definitely true. If true in some → maybe true.

## Example

**Statements:** All cats are dogs. All dogs are lions.

**Diagram:** Cat circle inside Dog circle inside Lion circle (nested)

**Conclusions:**
1. All cats are lions. ✓ (nested = definite)
2. Some lions are cats. ✓ (valid reverse)
3. Some dogs are cats. ✓ (valid reverse)

## Practice

**Statements:** Some A are B. All B are C.

What follows?
1. Some A are C? ✓ (Rule 3)
2. Some C are A? ✓ (also Rule 3)
3. All C are A? ✗ (can't convert "Some" to "All")
4. Some A are not C? ✗ (not definite)
