# Pattern Card: Syllogisms

## Use when:
- "All/No/Some A are B" statements → draw diagrams
- "Which conclusions follow?" → check each against ALL diagrams
- "Either-or" conclusions → same subjects, no definite proof
- "What can be inferred?" → apply Rule 1-6 table

## Core question:
"Given these premises, what MUST be true in every valid diagram?"

## Common mistakes:
- Using "converse" wrong: All A→B means "Some B are A" NOT "All B are A"
- Drawing only one diagram when multiple are possible
- Using real-world knowledge instead of the given statements
- Confusing "may be true" with "must be true"
- Missing complement pairs (either-or conclusions)

## Interview sentence:
"I'll draw Venn diagrams from each premise and test each conclusion against all possible arrangements."

## Time/Space Analysis:
- O(1) — diagram-based, formula lookup
