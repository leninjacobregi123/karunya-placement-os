# Answer Key

## Quiz Answers
### Q1
**Answer:** `Rs 450`
25% discount on Rs 600 = 0.25 × 600 = Rs 150 off. Selling price = 600 - 150 = Rs 450. Alternatively: 600 × 0.75 = Rs 450.

### Q2
**Answer:** `15%`
% change = ((57500 - 50000) / 50000) × 100 = (7500 / 50000) × 100 = 15%.

### Q3
**Answer:** `4% decrease`
Net change = 20 + (-20) + (20 × -20)/100 = 0 + (-4) = -4%. So it's a 4% decrease. Check: 100 → 120 → 96. Not back to 100!

### Q4
**Answer:** `68%`
(34/50) × 100 = 0.68 × 100 = 68%. Shortcut: 34/50 = 68/100 = 68% (multiply numerator and denominator by 2).
