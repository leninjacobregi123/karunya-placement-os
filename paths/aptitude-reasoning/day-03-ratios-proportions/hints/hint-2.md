When combining ratios (e.g., a:b and b:c), the golden rule is to make the common term equal using LCM. Also remember: a/b = c/d means you can cross-multiply as a x d = b x c. Practice with simple numbers first (a=2, b=4, c=3, d=6) to see the pattern visibly.

For mean proportion, always take the square root of the product — not the average. sqrt(16 x 81) = 36. This is because if a:b = b:c, then b^2 = a x c (from cross-multiplication), so b = sqrt(a x c).
