# Averages

## Learning Objectives

- Understand the arithmetic mean formula: sum / number of items
- Solve average speed problems using total distance / total time
- Apply averages to age problems and changing average scenarios
- Find missing numbers from given averages
- Work with weighted averages in real-world contexts

## 1. Arithmetic Mean — The Basics

**Average = Sum of all values / Number of values**

Real-world example (Test scores): Riya took 5 unit tests: 72, 85, 63, 91, 69.
- Sum = 380, n = 5, Average = 380 / 5 = 76

Real-world example (Temperatures): Chennai: 34, 36, 32, 38 °C over 4 days.
- Sum = 140, n = 4, Average = 35 °C

Tip: The average must lie between the smallest and largest value.

## 2. Average Speed

Average speed is **NOT** (speed1 + speed2) / 2. It is always:

**Average Speed = Total Distance / Total Time**

When equal distances at two speeds, use: (2 × x × y) / (x + y)

Round trip example: Chennai to Bangalore at 60 km/h, return at 40 km/h, 120 km each way.
- Total distance = 240 km, Time = 2 + 3 = 5 hours
- Average speed = 240 / 5 = 48 km/h
- (60 + 40) / 2 = 50 would be WRONG — more time spent at slower speed

## 3. Average Age Problems

Every person ages equally each year. When "n" people age "t" years, the average increases by t.

Family example: Three friends aged 18, 22, 26. Average = 66/3 = 22.
After 5 years: Each ages 5 years. New average = 22 + 5 = 27.

Inheriting: Average of 30 students is 14. Teacher joins, average becomes 14.5.
- Students sum = 420, Total (31 people) = 449.5
- Teacher's age = 29.5 years

## 4. Changing Average

When a new value joins: if new value > old average → average goes up.
Change = (New value − Old average) / New total count

Salary example: 10 employees average Rs 45,000. Manager joins, average rises Rs 2,000.
- Old sum = 450,000, New average = 47,000, New total = 11
- Manager's salary = 517,000 − 450,000 = Rs 67,000

## 5. Finding Missing Numbers

Formula: Missing = (Average × Count) − Sum of known

Example: Scores 78, 82, 90 give average 85 across 4 tests.
- Required total = 340, Known = 250, Fourth score = 90

## 6. Weighted Averages

Don't average averages from different group sizes!

Class sections: Section A (30 students, avg 72), Section B (40 students, avg 68).
- A total = 2,160, B total = 2,720
- Overall average = 4,880 / 70 = 69.71
