# Syllogisms — Completing Lesson

## Valid Inference Rules

1. **All + All → All**
   All A are B. All B are C. → All A are C.

2. **All + No → No**
   All A are B. No B are C. → No A are C.

3. **No + All → No (reversed)**
   No A are B. All B are C. → No C are A.

4. **No + Some → No (reversed)**
   No A are B. Some B are C. → No C are A (but Some C are not A)

5. **Some + All → Some (both directions)**
   Some A are B. All B are C. → Some A are C AND Some C are A.

6. **Some + No → Some (reversed)**
   Some A are B. No B are C. → Some A are not C.

7. **Some + Some → NO conclusion** (can't infer anything)

## Conversion Rules

| Original | Valid | Invalid |
|---|-|-|
| All A → B | Some B → A | All B → A |
| No A is B | No B is A | Some A are not B |
| Some A are B | Some B are A | All B are A |
| Some A are not B | Cannot convert | — |

## The Diagram Method

**Step 1:** Draw each premise on the SAME diagram.
**Step 2:** Test each conclusion against ALL valid diagrams.
**Step 3:** If true in ALL diagrams → definitely true. If in NONE → definitely false. If in SOME → uncertain.

### When Multiple Diagrams Exist

**Premises:** All A are B. Some B are C.

Diagram 1: C overlaps A (inside B)
Diagram 2: C outside A (but still in B)

"Is Some A C?" → Both possible, so answer is UNCERTAIN.

## Complements: The "Either-Or" Case

Two conclusions form a complement when:
1. Same subjects
2. One is "all/some" and other is "no/some-not"
3. Neither can be proved alone

Common pairs: "Some A are B" + "No A is B"

## Interview Strategy

1. Draw diagrams first — always
2. Check: are BOTH premises universal? If yes, conclusion always possible
3. Any "some" or "no" premise? Be cautious
4. Always check reversals are valid
5. For complements, if both seem equally valid, it's either-or
